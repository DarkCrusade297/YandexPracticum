# YandexPracticum
Прохождение обучения

Для того, чтобы выполнить сборку проекта используйте команду "dotnet build" Для запуска проекта используйте команду "dotnet run"

После запуска приложение сваггер находится на порту 5182

GET /events - получение списка всех мероприятий POST /events - создание мероприятия GET /events/{id} - получение мероприятия по его уникальному идентификатору PUT /events/{id} - изменение данных о мероприятии по его уникальному идентификатору DELETE /events/{id} - удаление мероприятия по его уникальному идентификатору
